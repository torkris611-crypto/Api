# Менеджер заметок

Веб-приложение для управления заметками с использованием AJAX (Fetch API).

## Функционал
- Просмотр, создание, редактирование и удаление заметок
- Поиск по заголовку и содержанию
- Фильтрация по категориям
- Загрузка изображений
- Бесконечная прокрутка
- Уведомления об операциях

## Запуск проекта

### 1. Установите JSON Server

npm install -g json-server

### 2. Запустите сервер
В папке с проектом выполните:


json-server --watch db.json --port 3000

Сервер будет доступен по адресу: http://localhost:3000/notes
